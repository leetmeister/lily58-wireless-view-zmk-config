# Custom Lily58 Wireless ZMK Firmware Config and Keyboard Layout

## Configuration

This is a fork of [typeractive.xyz's](https://docs.typeractive.xyz/build-guides/lily58-wireless) Lily58 wireless keyboard [ZMK config repo](https://github.com/typeractivexyz/lily58-wireless-view-zmk-config). As with other ZMK config repos, this fork runs the standard ZMK firmware build Github Action that is triggered on every push.

Some configuration details of note:

- Mouse keys support is enabled with higher than default movement and scroll values.
- Customized animated image for the Nice!View on the right side via customized [https://github.com/leetmeister/nice-view-mod](https://github.com/leetmeister/nice-view-mod).
- ZMK Studio enabled and unlocked by default.

## Keymap

The graphical representation below is directly generated from the keymap file using [Cem Aksoylar's](https://github.com/caksoylar) excellent [keymap-drawer](https://github.com/caksoylar/keymap-drawer) tool.
